# Quiz Game System (C++)

## 1. Introduction
This project is a simple Quiz Game System developed using C++. It allows users to answer questions and shows the final score.

## 2. Objectives
- To implement OOP concepts
- To build a quiz system
- To improve coding skills

## 3. Methodology
The system asks multiple-choice questions and checks answers.

## 4. Implementation
The program is written in C++ and runs in console.

## 5. Conclusion
This project helps understand C++ and OOP concepts.
